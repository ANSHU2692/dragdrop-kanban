## 📌 What is Kanban?

**Kanban** is a visual workflow management method used to organize and track tasks efficiently.

It was originally developed by Taiichi Ohno at Toyota to improve manufacturing processes.

### 🧠 Core Idea:

> Visualize your work → Limit overload → Improve flow

### 📊 Basic Workflow:

- 📝 **To Do** → Tasks yet to start
- 🚧 **In Progress** → Tasks currently being worked on
- ✅ **Done** → Completed tasks

This project is a direct implementation of these Kanban principles in a web-based interface.

## ✨ Features

- 📝 Create tasks with title & description
- 🔄 Drag & Drop tasks across columns
- 📊 Real-time task count tracking
- 💾 Persistent storage using localStorage
- 🗑️ Delete tasks instantly
- 🎨 Clean glassmorphism UI with smooth animations
- 📱 Responsive and minimal design

---

## 🧱 Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling (Glassmorphism + Animations)
- **JavaScript (Vanilla)** – Logic, State & Interactions

---

---

## ⚙️ How It Works

- Tasks are dynamically created using JavaScript
- Drag & Drop is implemented via the **HTML Drag and Drop API**
- Application state is managed using JavaScript objects
- Tasks are stored in **localStorage**, ensuring persistence after reload
- On reload, tasks are fetched and rendered automatically
